Das "Cityline.MCB32.1.2.0.pack" Pack vereinfacht den Umgang mit der TouchP0P1 Bibliothek.

Das Pack wird mit einem Doppelklick installiert oder in Keil im Packinstaller mit dem Menupunkt "Import".

- Wenn das Pack installiert ist, kann in Runtime-Manager das Board (MCB Rev C oder D) ausgewählt werden.
- Unter BoardSupport->I/O kann die TouchP0P1 Bibliothek dem Projekt hinzugefügt werden.
- Ein Klick auf die TouchP0P1 Bezeichnung öffnet das PDF-Manual
- Wird mit "Add New Files" eine neue Datei hinzugefügt, so kann im Dialog 
  unter "User Templates"->Board Support ein Testprogramm  hinzugefügt werden.
  
Das Pack ist *experimentell* und keine offizielle Veröffentlichung.
Es wird keinerlei Wartung oder Garantie übernommen.
